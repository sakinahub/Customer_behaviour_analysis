# Customer_behaviour_analysis
🛍️ Customer Shopping Behavior Analysis
Overview

This project analyzes customer shopping behavior data to uncover purchasing patterns, customer preferences, spending trends, and business insights. The analysis was performed using Python, SQL, and Power BI to transform raw customer transaction data into actionable insights for decision-making.

Project Workflow
1. Data Loading
Imported customer shopping dataset into Python.
Inspected data structure and column information.
2. Data Cleaning
Checked for missing values.
Removed duplicates.
Validated data types.
Ensured data consistency.
3. Exploratory Data Analysis (EDA)

Performed analysis on:

Customer demographics
Spending behavior
Product category preferences
Seasonal purchasing trends
Subscription vs Non-subscription customers
Payment method usage
Review ratings distribution
4. SQL Analysis

Executed SQL queries to answer business questions such as:

Which product categories generate the highest revenue?
Which locations have the highest sales?
What is the average purchase amount by gender?
How do subscription customers behave compared to non-subscribers?
Which payment methods are most commonly used?
5. Dashboard Development

Created an interactive Power BI dashboard featuring:

Total Revenue
Average Purchase Amount
Category-wise Sales
Customer Demographics
Seasonal Trends
Subscription Analysis
Location-Based Performance
6. Reporting & Presentation
Documented key findings in a project report.
Developed a professional presentation using Gamma AI.
